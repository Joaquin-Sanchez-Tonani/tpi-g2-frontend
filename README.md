# TPI G2 Frontend

Frontend project for TPI, built with **Vite + React**.

---

## Fork & Clone

Fork the repo:  
[GitHub repo](https://github.com/Joaquin-Sanchez-Tonani/tpi-g2-frontend)  

Clone it locally:

```bash
git clone https://github.com/YOUR-USERNAME/tpi-g2-frontend.git
cd tpi-g2-frontend
npm install  # OR npm i
npm run dev
```
